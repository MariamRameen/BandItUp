## **Project Overview**

BandItUp is an **AI-powered IELTS preparation web application** that helps learners improve all four core skills: **Speaking, Listening, Reading, and Writing**.

The platform provides **adaptive learning**, **real-time AI feedback**, and **progress tracking** to make preparation smarter, personalized, and efficient.

---

## **Modules Overview**

**1. Speaking Practice & Pronunciation Feedback**  
Structured IELTS speaking practice (Parts 1–3) and free-speaking mode. Includes phoneme-level pronunciation feedback, grammar/vocabulary evaluation, and band prediction.

**2. Listening Practice with Adaptive Difficulty**  
Timed audio passages with UK, US, and AU accents. AI-generated questions, feedback, and adaptive difficulty.

**3. Reading Practice**  
Timed IELTS-style passages with adaptive difficulty and detailed feedback. Includes micro-practice drills.

**4. Writing Practice**  
Task 1 & 2 exercises (Academic/General). AI evaluates task achievement, coherence, grammar, and vocabulary.

**5. Vocabulary & Grammar Trainer**  
Band-wise vocabulary packs and grammar exercises. Tracks progress and suggests improvements.

**6. Baseline Test**  
Initial mock test to establish user band score and generate personalized study plan.

**7. Mock Tests Generator**  
Weekly mock tests with AI-evaluated scores and targeted feedback.

**8. Study Planner & Notification System**  
Personalized study schedules, reminders, and activity tracking.

**9. User Dashboard & Progress Tracker**  
Centralized progress view: modules, tests, strengths, weaknesses, and predicted band.

**10. User Login & Profile System**  
Secure login/signup (email, phone, Google). Profile customization including avatar, theme, and target band.

**11. Admin Dashboard**  
User management, subscription overview, content control, and moderation tools.

**12. Live Chat & Support System**  
Real-time support via live chat and AI chatbot for FAQs and user issues.

## **Usage Instructions**

### **Frontend (React 5173)**

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

Your frontend will run at:
```
http://localhost:5173
```

---

### **Backend (Node.js + Express)**

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Start the backend server:
```bash
node app.js
```
